# AGAD Department Form Templates

This file is a lightweight placeholder for the department-specific form templates used by the AGAD Document Generator Agent.

The actual templates are stored in `references/department-field-requirements.json`.

## Department mapping

- `Admitting/Registration`
- `Insurance/HMO Desk`
- `Emergency/Triage`
- `Billing/Discharge`

Each template is a plain text document with placeholder tokens matching keys from the intake slots.
